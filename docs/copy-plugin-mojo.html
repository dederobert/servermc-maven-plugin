servermc:copy-plugin

Full name:

fr.lehtto.maven.plugins:servermc-maven-plugin:0.1.0-SNAPSHOT:copy-plugin

Description:

Copies plugins to server plugins folder. Copies the current plugin and defined additionalPlugins.

Attributes:

  • Requires a Maven project to be executed.
  • Requires dependency resolution of artifacts in scope: compile.
  • Since version: 0.0.1.

Required Parameters

Name Type Since Description
<skipAdditionalPlugins> boolean 0.0.1 Whether the additional plugins should be resolved and copied.
Default value is: false.
User property is: skipAdditionalPlugins.

Optional Parameters

Name Type Since Description
<additionalPlugins> List 0.0.1 The list of additional plugins to copy.
User property is: additionalPlugins.
<jarName> String 0.0.1 The jar name.
Default value is: ${project.build.finalName}.
User property is: jarName.
<serverDirectory> File 0.0.1 The target server's directory.
Default value is: target/server.
User property is: serverDirectory.
