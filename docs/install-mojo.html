servermc:install

Full name:

fr.lehtto.maven.plugins:servermc-maven-plugin:0.0.2-SNAPSHOT:install

Description:

Installs (download, setup) the server.

Attributes:

  • Requires a Maven project to be executed.
  • Since version: 0.0.1.

Required Parameters

Name Type Since Description
<mcVersion> String 0.0.1 Minecraft version to use.
User property is: mc-version.
<overrideJar> boolean 0.0.1 Whether the JAR file has to be overridden if it exists.
Default value is: false.
User property is: override-jar.
<overrideProperties> boolean 0.0.1 Whether the JAR file has to be overridden if it exists.
Default value is: false.
User property is: override-properties.

Optional Parameters

Name Type Since Description
<baseUrl> String 0.0.1 The base URL to use to download paper MC.
Default value is: https://api.papermc.io/v2/projects/paper/.
User property is: base-url.
<buildNumber> String 0.0.1 The build number to use.
User property is: buildNumber.
<eula> boolean 0.0.1 Whether the EULA is agreed.
Default value is: false.
User property is: eula.
<properties> Map 0.0.1 Initial values of "server.properties".
User property is: properties.
<serverDirectory> File 0.0.1 The target server's directory.
Default value is: target/server.
User property is: serverDirectory.
