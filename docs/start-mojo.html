servermc:start

Full name:

fr.lehtto.maven.plugins:servermc-maven-plugin:0.1.0-SNAPSHOT:start

Description:

Starts the Minecraft server.

Attributes:

  • Requires a Maven project to be executed.
  • Since version: 0.0.1.

Required Parameters

Name Type Since Description
<remoteDebug> boolean 0.0.1 Prepare the sever to attach a remote debug.
Default value is: false.
User property is: remoteDebug.

Optional Parameters

Name Type Since Description
<debugPort> int 0.0.1 The port to use for remote debugger.
Default value is: 5005.
User property is: debugPort.
<memoryMax> int 0.0.1 The maximum size, in Go, of memory allocation pool (-Xms).
Default value is: 2.
User property is: memoryMax.
<memoryMin> int 0.0.1 The minimum size, in Go, of memory allocation pool (-Xms).
Default value is: 1.
User property is: memoryMin.
<serverDirectory> File 0.0.1 The target server's directory.
Default value is: target/server.
User property is: serverDirectory.
